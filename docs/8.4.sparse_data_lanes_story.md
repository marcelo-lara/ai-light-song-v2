# EPIC 8 - Story 8.4

## Sparse Data Lanes

## Goal

Render lower-density inference surfaces such as sections, symbolic phrase windows, chords, pattern occurrences, and event windows.

## Inputs

- `data/artifacts/<Song - Artist>/layer_b_symbolic.json`
- `data/artifacts/<Song - Artist>/layer_a_harmonic.json`
- `data/artifacts/<Song - Artist>/layer_d_patterns.json`
- `data/artifacts/<Song - Artist>/section_segmentation/sections.json`
- `data/artifacts/<Song - Artist>/event_inference/events.machine.json`

## Acceptance Criteria

- Sparse lanes render directly from artifact timing windows.
- The symbolic phrases lane reads `phrase_windows[]` from `layer_b_symbolic.json` and displays all windows grouped by `phrase_group_id`.
- Symbolic phrase windows stack into multiple rows when overlaps occur, using the same compaction logic as machine event windows.
- Clicking or selecting a region affects only browser playback/navigation state.
- No region interaction writes state into generated-data directories.
