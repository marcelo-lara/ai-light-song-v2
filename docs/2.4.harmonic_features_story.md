# EPIC 2 - Story 2.4

## Harmonic Features

## Goal

Derive harmonic descriptors that are useful to both human interpretation and downstream lighting logic.

## Inputs

- Chord events from Story 2.2
- Key estimate from Story 2.3
- HPCP prominence data from Story 2.1
- Section windows from Story 4.2 when section summaries are needed

## Dependencies

- Rule-based cadence detection
- Heuristics for tension, clarity, mobility, and harmonic-role labeling

## Processing Steps

1. Extract chord roots and chord qualities from decoded chord labels.
2. Measure harmonic mobility from chord-change frequency and event turnover.
3. Detect cadence candidates using chord transitions and resolution heuristics.
4. Compute harmonic tension scores using dissonance, motion, ambiguity, or unresolved transitions.
5. Summarize harmonic features globally and by section.
6. Persist the result into the harmonic layer contract.

## Output Artifacts

- enriched `layer_a_harmonic.json` containing summaries, section cards, cadence notes, and tension peaks

## Output Schema

```json
{
  "harmonic_features": {
    "tension": 0.70,
    "cadence": "open_cadence",
    "harmonic_mobility": 1.0,
    "harmonic_clarity": 0.18
  }
}
```

## Acceptance Criteria

- Harmonic summaries are readable by a musician.
- Section-level harmonic cards align with the decoded chord events.
- Tension values discriminate calm passages from unresolved or dense passages.

## Validation

- Spot-check cadence labels against obvious transitions.
- Confirm section summaries reference only events inside the section window.
- Compare repeated pattern sections for consistent feature values.

## Failure Cases

- Cadence detection overfires on repetitive loops
- Tension scores are dominated by noise rather than real harmonic change
- Section summaries are copied rather than recomputed from section-local evidence

## Out of Scope

- Full functional harmony parsing
- Genre-specific harmonic semantics beyond the shared descriptor set, which should rely on Story 2.5 genre guidance when available
