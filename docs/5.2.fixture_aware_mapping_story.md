## ⚙️ Tasks

---

### 1. Load Fixture Configuration

Parse fixtures.json:

- id
- fixture type
- base_channel
- location (x,y,z)

---

### 2. Classify Fixtures into Roles

#### Derived Roles

| Role | Fixtures |
|------|---------|
| moving_head_main | head_el150 |
| moving_head_fx | mini_beam_prism_l, mini_beam_prism_r |
| parcan_inner | parcan_l, parcan_r |
| parcan_outer | parcan_pl, parcan_pr |

Rules:
- center → main
- edges → outer
- symmetric → FX

---

### 3. Map Features → Lighting Dimensions

| Feature | Dimension |
|--------|----------|
| loudness | intensity |
| spectral_flux | motion |
| onset_strength | accents |
| note_density | complexity |
| section_type | scene |

Normalize all to 0–1

---

### 4. Fixture-Aware Behavior Mapping

#### moving_head_main (head_el150)

- pan = melodic contour
- tilt = register
- speed = spectral_flux

---

#### moving_head_fx (mini beams)

- strobe on strong onset
- fast sweeps in chorus

---

#### parcan_inner

- dimmer = bass pulse
- color = harmonic key

---

#### parcan_outer

- color = section color
- dimmer = smoothed loudness

---

### 5. Section-Based Scene Logic

#### Intro
- low intensity
- slow movement

#### Verse
- rhythmic pulses
- moderate motion

#### Chorus
- full intensity
- strobes + fast motion

#### Bridge
- contrast + reduction

---

### 6. Generate DMX Events

Example:

{
  "time": 60.0,
  "fixtures": {
    "head_el150": {"pan": 120, "tilt": 45},
    "parcan_l": {"dimmer": 0.9}
  }
}
