# EPIC 8 - Story 8.6

## Semantic Zoom and Performance Guardrails

## Goal

Protect the debugger from unbounded rendering cost while preserving useful inference visibility across zoom levels.

## Acceptance Criteria

- Zoom state is shared across lanes.
- Dense artifact views can cluster or summarize data when zoomed far out.
- Rendering policy stays browser-local and does not create new generated files.
