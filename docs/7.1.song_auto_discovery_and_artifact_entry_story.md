# EPIC 7 - Story 7.1

## Song Auto-Discovery and Artifact Entry

## Goal

Allow the internal debugger to discover available per-song artifact directories automatically so users do not need to type song folder names manually.

## Inputs

- `data/artifacts/`
- `data/output/` as optional helper context

## Processing Steps

1. Discover per-song directories directly from `data/artifacts/`.
2. Present those directories as selectable debugger entries.
3. Initialize debugger loading from a selected discovered song directory.
4. Preserve read-only access to all mounted generated-data paths.

## Output

- browser-only song selection and discovery state inside the debugger

## Acceptance Criteria

- Users can choose a discovered song directory without typing it manually.
- Song discovery reads the mounted artifact tree directly.
- Discovery state remains browser-local and read-only.
- No debugger-generated files are written into `data/artifacts/` or `data/output/`.
