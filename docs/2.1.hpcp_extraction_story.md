# EPIC 2 - Story 2.1

## HPCP Feature Extraction

## Goal

Extract beat-aligned chroma features from the harmonic stem to support key, chord, and harmonic-motion inference.

## Inputs

- Harmonic stem from Story 1.1
- Canonical beat grid from Story 1.2

## Dependencies

- Essentia HPCP extraction
- Tuning correction and chroma normalization

## Processing Steps

1. Load the harmonic stem.
2. Apply tuning estimation and tuning correction when required.
3. Compute frame-level HPCP features.
4. Aggregate or summarize HPCP vectors per beat using the canonical beat grid.
5. Persist both high-resolution and beat-aligned representations if needed by later stories.

## Output Artifacts

- beat-aligned HPCP feature file under `data/artifacts/<Song - Artist>/`
- references to that file in `info.json` and `layer_a_harmonic.json`

## Output Schema

```json
{
  "schema_version": "1.0",
  "song_id": "Song - Artist",
  "generated_from": {
    "harmonic_stem": "data/stems/Song - Artist/harmonic.wav",
    "beats_file": "data/artifacts/Song - Artist/essentia/beats.json"
  },
  "hpcp_by_beat": [
    {
      "beat": 1,
      "time": 0.512,
      "vector": [0.02, 0.10, 0.42, 0.18, 0.04, 0.01, 0.00, 0.08, 0.25, 0.30, 0.05, 0.03]
    }
  ]
}
```

## Acceptance Criteria

- Chroma representation is stable enough to support expected chord and key inference.
- Beat aggregation does not destroy obvious tonal motion.
- Output can be reused without recomputing raw HPCP in every downstream story.

## Validation

- Spot-check dominant pitch classes against known chord centers.
- Confirm feature continuity across adjacent beats in stable passages.
- Compare chord and key downstream outputs against validation-only reference files when they are available, not the raw HPCP itself.
- Reference files are optional, read-only validation inputs and must never overwrite inferred harmonic outputs.

## Failure Cases

- Noisy harmonic stem causes unstable chroma vectors
- Tuning mismatch distorts pitch-class prominence
- Beat aggregation windows misalign with true harmonic rhythm

## Out of Scope

- Learned chroma embeddings
- Melody extraction from HPCP
